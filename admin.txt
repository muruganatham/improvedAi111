Q7: Course allocation
Q: how many courses allocated?
A:
## Allocated Courses Count

**Total Allocated Courses:** **1,486**

This count represents the number of courses that have been allocated to academic programs through the `course_academic_maps` table. These are courses that have been assigned to specific colleges, departments, batches, and sections for student enrollment.

**Note:** This is different from:
- **Available courses** (in the `courses` table with status = 1) - these are all courses in the catalog
- **Enrolled courses** (in `course_wise_segregations` or `user_course_enrollments`) - courses that students have actually enrolled in

The allocated courses represent the bridge between available courses and student enrollment - they're the courses that have been made available to specific academic groups.

[SQL Executed]:
SELECT COUNT(*) as allocated_courses_count FROM course_academic_maps WHERE status = 1;
